Installing Classy

The recommended method for integrating Classy into your project is using cocoapods

Add Classy to your Podfile

{% highlight ruby %}pod 'Classy'{% endhighlight %}

Run pod install from your command line

{% highlight bash %}$ pod install{% endhighlight %}
